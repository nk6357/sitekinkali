# Система управления товарами через Menu12

## Как добавить товар

Система автоматически загружает товары из папки `Menu12/` при сборке проекта.

### Структура папок

```
Menu12/
├── Холодные закуски/
│   └── [название товара]/
│       ├── 1.txt (обязательно)
│       └── [фото].[jpg|png|jpeg|webp] (опционально)
├── Горячие закуски/
├── Супы/
└── [другие категории]/
```

### Пример добавления товара

1. **Создайте папку товара** в соответствующей категории:
   ```
   Menu12/Супы/Борщ украинский/
   ```

2. **Создайте файл `1.txt`** с информацией товара (3 строки):
   ```
   250
   Борщ украинский
   Борщ с говядиной, овощами и сметаной
   ```
   
   - Строка 1: Цена (число)
   - Строка 2: Название товара
   - Строка 3: Описание товара

3. **Добавьте фото** товара (опционально):
   - Любой формат: `.jpg`, `.png`, `.jpeg`, `.webp`
   - Любое имя файла
   - Например: `dish.jpg` или `photo.png`

4. **Пересоберите проект**:
   ```bash
   npm run build
   ```

## Процесс генерации

При запуске `npm run build` выполняется:

1. **`npm run generate`** - сканирует Menu12/ и генерирует `src/data/menu.ts`
2. **`tsc -b`** - компилирует TypeScript
3. **`vite build`** - собирает приложение

## Требования

- Название категории должно точно совпадать с названием в списке выше
- Файл `1.txt` обязателен для каждого товара
- Цена должна быть числом (целое число)
- Фото опционально (если нет - будет показано место для картинки)
